# Poker game
## 專案介紹
這是一個使用 C# Windows Forms 開發的 Poker 應用程式。本專案除了具備基本的下注功能，以及簡單好上手的遊戲模式，自動判斷排型。
## 核心功能
* **下注系統：** 玩家可自訂押注金額，系統會自動扣除資金，並在結算時依照賠率發放獎金。
* **發牌與洗牌：** 模擬真實洗牌邏輯，隨機發放 5 張撲克牌。
* **換牌機制：** 玩家可點擊牌面選擇保留或替換的牌，並進行一次換牌。
* **牌型判斷：** 系統能精準判斷同花大順、同花順、鐵支、葫蘆、同花、順子、三條、兩對、一對及雜牌。
## 執行說明
1. 將專案下載或 Clone 至本機。
2. 使用 Visual Studio 開啟 `Poker.sln` 方案檔。
3. 按下 `F5` 或點擊「開始」編譯並執行專案。
4. **遊戲流程：**
   - 在「押注金額」欄位輸入想下注的金額。
   - 點擊 **[押注]** 按鈕（系統會扣除總資金）。
   - 點擊 **[發牌]** 按鈕。
   - 點擊想要替換的牌（牌面會蓋上表示將被替換）。
   - 點擊 **[換牌]** 獲取新牌。
   - 點擊 **[判斷牌型]** 進行結算，系統將顯示結果並發放獎金。
